## 一、你需要准备什么

1. 一台 Windows 10/11 电脑
2. 一个 QQ 号（给机器人用，建议小号）
3. 一个 DeepSeek API Key（https://platform.deepseek.com 注册充值10元）
4. Python 3.12（从 https://python.org 下载，安装时勾选"Add to PATH"）

【重要】不要从微软商店装Python！从python.org下载！

## 三、安装步骤（按顺序来）

### 第1步：安装 Python
1. 运行 python-3.12.x-amd64.exe
2. 勾选底部 "Add Python to PATH"（非常重要！）
3. 点击 Install Now
4. 安装完成后，打开 CMD 验证：
   python --version
   应该显示 Python 3.12.x

### 第2步：安装 Python 依赖包
打开 CMD，输入：
   python -m pip install websockets requests

如果报错"pip不是内部命令"，说明Python没装好或没勾选Add to PATH。
解决方法：重装Python，确保勾选"Add Python to PATH"。

### 第3步：安装 NapCatQQ
1. 解压 NapCat.Shell.zip 到某个目录（例如 D:\NapCat）
2. 双击 NapCatWinBootMain.exe
3. 会弹出一个黑窗口，显示QQ登录二维码
4. 用手机QQ（机器人号）扫码登录
5. 看到 "NapCat4 Is Running" 就是成功了

验证：浏览器打开 http://127.0.0.1:3000/get_status
看到 "online":true 就对了。

### 第4步：配置百约
1. 用记事本打开 bot.py
2. 找到第28行：DEEPSEEK_KEY = ""
3. 填入你的 DeepSeek API Key（在 platform.deepseek.com 获取）
4. 找到第35行：OWNER_QQ = "3154997545"
5. 改成你自己的 QQ 号（主人号，机器人会对这个号用女友模式）
6. 保存文件

### 第5步：配置 NapCat WebSocket
1. 浏览器打开 NapCat WebUI：
   在 NapCat 启动窗口里找到 "WebUi Token: xxxxxxxx"
   然后访问 http://127.0.0.1:6099/webui?token=你的Token
2. 找到「网络配置」→「WebSocket客户端」
3. 点击「添加」：
   - 名称：百约
   - URL：ws://127.0.0.1:8001
4. 保存

### 第6步：启动百约
打开 CMD，进入百约文件夹：
   cd 百约文件夹路径
   python bot.py

看到 "NapCat 已接入" 就是成功了！

然后用你的主人QQ号给机器人QQ号发消息，百约就回你了。

## 四、常见问题排查

| 问题 | 原因 | 解决方法 |
|---|---|---|
| python不是内部命令 | Python没装好 | 重装Python，勾选Add to PATH |
|---|---|---|
| pip不是内部命令 | 同上 | 用 python -m pip 代替 pip |
| 连不上NapCat(端口3000) | NapCat没启动或QQ没登录 | 先开NapCat，扫码登录QQ |
| 端口8001被占用 | 旧的百约/其他程序占着 | 任务管理器关掉python.exe，或等v2.0自动换端口 |
| 机器人不回消息 | WebSocket没配好 | 检查NapCat WebUI的WS地址是 ws://127.0.0.1:8001 |
| AI回复很慢 | DeepSeek服务器忙 | 正常现象，稍等几秒 |
| 机器人回复很AI/像客服 | 人格设定不够具体 | 改bot.py里的get_system_prompt函数 |
| 群聊里不理人 | 需要@机器人才触发 | 在群里@机器人名字 |

## 五、自定义人格

打开 bot.py，找到 get_system_prompt 函数，修改里面的文字即可。

百约有两套人设：
- is_owner = True  → 对你（主人QQ号）的人格 → AI女友模式
- is_owner = False → 对其他人的人格 → 酷酷AI助手模式

你越具体，她越像真的。

## 六、怎么让机器人发表情

百约内置了QQ表情翻译系统，直接在人格设定里告诉它能用哪些表情：
[爱心] [笑哭] [呲牙] [调皮] [偷笑] [坏笑] [酷] [好的] [吃瓜]
[点赞] [抱拳] [玫瑰] [发呆] [亲亲] [害羞] [无语] [叹气] [睡] [惊讶]

原理：你发QQ表情 → 代码翻译成文字给AI → AI回复文字表情 → 代码翻译成QQ表情发出去

## 七、怎么更新到最新版本

1. 去 GitHub 下载最新 bot.py：https://github.com/bailidashi/baiyue
2. 替换旧文件
3. 重新填入你的 API Key 和 QQ号
4. 重启 python bot.py

## 九、技术架构（给懂的人看）

QQ消息 → NapCatQQ(OneBot v11) → WebSocket(8001) → bot.py → DeepSeek API → NapCat HTTP(3000) → QQ回复

技术栈：
- Python 3.12
- websockets（WebSocket服务器）
- requests（HTTP客户端）
- DeepSeek API（AI大脑）
- NapCatQQ（QQ协议适配，OneBot v11标准）
- 双向QQ表情CQ码翻译
- 长期记忆+自动摘要系统
