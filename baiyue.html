🤖 从零搭建你的 QQ AI 机器人

百约 · BaiYue 完整教程 — Python + NapCatQQ + DeepSeek

Python 3.12 NapCatQQ DeepSeek API OneBot v11 WebSocket

🧠 原理：你的机器人是怎么运行的

你在QQ发消息
  ↓
NapCatQQ 拦截消息 → 通过 WebSocket 转发给 Python
  ↓
Python 把消息发给 DeepSeek（AI大脑）
  ↓
DeepSeek 生成回复 → Python 发回 NapCat → QQ
  ↓
你看到回复
角色负责什么
NapCatQQ翻译官。把QQ消息翻译成Python能读的JSON格式
Python程序中间人。收消息、调AI、发回复
DeepSeek API大脑。真正"说话"的那个

📦 第一步：安装 Python

  1. 打开 python.org，下载 Python 3.12.x
  2. 安装时勾选 "Add Python to PATH"
  3. 验证：python --version 应显示 Python 3.12.x
  4. 装依赖：python -m pip install websockets requests

🔌 第二步：安装 NapCatQQ

  1. 下载：github.com/NapNeko/NapCatQQ/releases
  2. 下载 NapCat.Shell.zip，解压到某目录（如 D:\NapCat）
  3. 双击 NapCatWinBootMain.exe
  4. 弹出二维码 → 用手机QQ（机器人号）扫码登录
  5. 验证：浏览器打开 http://127.0.0.1:3000/get_status，看到 "online":true

🔑 第三步：获取 DeepSeek API Key

  1. 打开 platform.deepseek.com
  2. 注册 → API Keys → 创建 Key → 复制保存
  3. 充值 10 块钱，够用几个月

💰 费用参考：聊100条约0.3元，聊一个月不到20块。

💻 第四步：写代码

完整代码在 github.com/bailidashi/baiyue，核心结构：

bot.py
├── 配置（API Key / QQ号 / 端口）
├── 人格设定（女友模式 / 助手模式）
├── AI大脑（调 DeepSeek API）
├── QQ收发（通过 NapCat HTTP API）
├── 记忆系统（每人一个JSON文件）
├── 表情翻译（CQ码 ↔ 文字表情）
└── WebSocket 服务器（收消息）

🔗 第五步：配置 NapCat WebSocket

  1. 浏览器打开 NapCat WebUI（http://127.0.0.1:6099/webui?token=你的Token）
  2. 「网络配置」→「WebSocket客户端」→「添加」
  3. 名称：随便填 | URL：ws://127.0.0.1:8001
  4. 保存

🚀 第六步：启动

python bot.py

看到 NapCat 已连接 就成功了！去 QQ 上给机器人发消息试试。

🩺 常见问题

现象原因解决
连不上 NapCatNapCat 没启动/QQ没登录先启动NapCat扫码
不回消息WebSocket没配好检查WebUI的WS地址
端口占用旧机器人没关任务管理器杀python.exe
回复很慢DeepSeek服务器忙正常，等一下
回复像客服人格设定太弱改 system_prompt 函数

🎨 进阶：自定义人格

改 bot.py 里的 get_system_prompt 函数：

def get_system_prompt(is_owner, name):
    if is_owner:
        return f"""这是对你（主人）的人格设定。
        性格、说话风格、口头禅，越具体越像真的"""
    else:
        return """对其他人的人格设定。"""

人格示例

# 高冷猫娘
"你是一只猫娘，用喵结尾，高冷但对主人粘人喵。对陌生人冷淡。"

# 热血兄弟
"你是死党，热血笨蛋类型。永远支持他，会骂醒迷茫的他。"

# 毒舌吐槽役
"毒舌AI，幽默吐槽一切。不伤人，善于发现逻辑漏洞，偶尔自黑。"

😎 进阶：让机器人能发表情

百约内置了表情翻译系统：

  • 收：QQ表情 [CQ:face,id=66] → 翻译成 [爱心] 给AI看
  • 发：AI回复 [爱心] → 翻译回 [CQ:face,id=66] 发到QQ
